# Laravel UKK 2024 Project

## Description
This project was developed as part of the UKK assessment using Laravel framework.

## Tech Stack
- Laravel
- PHP
- MySQL
- Bootstrap
- Vite

## Installation

1. Clone repository
2. Run:
   composer install
3. Copy .env.example to .env
4. Configure database
5. Run:
   php artisan key:generate
   php artisan migrate
6. Run:
   npm install
   npm run dev
7. Start server:
   php artisan serve
